# moodHut
心情小屋
心情小屋是一款圖標導向日記 Android App，還提供了日記分享平台，使用者可利用此平台匿名與其他使用者互動，建立一對一匿名即時聊天室。

Android App (Kotlin) 作品 

圖標導向心情日記本結合匿名交友平台 : 

-前後端分離開發 

-前端：Android 

-後端：Node.js 

-前後端API串接：JSON 

-資料庫：MySQL



功能： 

-圖標導向心情日記本 

-統計心情與活動相關性 

-分享日記交流平台 

-匿名一對一聊天室
